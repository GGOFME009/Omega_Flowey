# 🌻 Omega Flowey Simulation
> "You really ARE an idiot."

An experimental web-based simulation recreating the chaotic atmosphere and visual glitches of the **Omega Flowey** (Photoshop Flowey) encounter from **Undertale**.

---

## 🎮 [Live Demo](https://ggofme009.github.io/Omega_Flowey/)
*(Make sure GitHub Pages is enabled in your repository settings to activate this link)*

---

## 🛠️ Technical Features
- **Glitch Engine:** Real-time visual distortions and screen shakes powered by Vanilla JavaScript.
- **Photoshop Aesthetics:** Replicating the distinct, uncanny art style of the original boss fight.
- **Dynamic Souls Logic:** Interactive elements mimicking the "Soul" mechanics within the HTML5 Canvas.
- **Unpredictable Animations:** Non-linear movement patterns that break the standard game boundaries.

## 🚀 Installation & Usage
To run the simulation locally:
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Open the project folder:
   ```bash
   cd Omega_Flowey
   ```
3. Run `index.html` in any modern web browser.

## 💻 Tech Stack
- **Structure:** HTML5 (Semantic Markup)
- **Logic:** Vanilla JavaScript (ES6+)
- **Styling:** CSS3 (Keyframe Animations & Filter Effects)
